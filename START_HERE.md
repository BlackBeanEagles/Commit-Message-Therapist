## 🆘 TROUBLESHOOTING

### "npm run dev doesn't work"
```bash
cd C:\Users\hridy\Desktop\win
npm install
npm run dev
```

### "App loads but no diagnoses showing"
1. Check browser console (F12)
2. Verify ANTHROPIC_API_KEY in .env.local
3. Try fallback mode (works without API key)

### "Analysis takes too long"
- Normal: 30-60 seconds on localhost
- Large repos may take longer
- Live on Vercel might be faster with caching

### "Featured diagnoses not showing"
1. Click "Analyze" tab
2. Don't enter anything yet
3. Scroll down
4. Should see 3 example cards

### "Mobile looks bad"
1. Open DevTools (F12)
2. Click responsive icon
3. Select iPhone view
4. Should be readable

### "Deploy to Vercel failed"
1. Check GitHub repo is public
2. Check you have Git installed
3. Check .env.local is in .gitignore
4. Try deploying again
